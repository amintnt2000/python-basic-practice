# Python Mini Projects

This repository contains two small Python projects designed to strengthen core programming skills. Each project focuses on a different fundamental concept such as string processing, loops, dictionaries, and user interaction. These projects are simple, beginner-friendly, and useful for practicing real-world logic in Python.

---

## 1. Text Analyzer

### Description
The Text Analyzer program receives a text input from the user and extracts key information from it. It calculates the total number of characters, counts the number of words, and identifies the most repeated word in the text. This project is useful for practicing string manipulation, loops, and dictionary-based frequency counting.

### Features
- Counts total characters in the text
- Counts total words using split()
- Builds a frequency dictionary for all words
- Detects the most repeated word
- Returns results in a clean dictionary format

---

## 2. Queue System

### Description
The Queue System simulates a simple real-life queue. The user can add people to the queue, serve the next person, display the current queue, or exit the program. This project helps practice list operations, condition handling, and interactive loops.

### Supported Commands
- add <name> : Adds a new person to the queue
- next : Removes and prints the first person in line
- show : Displays the current queue
- exit : Ends the program

### Features
- Queue behavior implemented using Python lists
- Safe handling of empty queues
- Clear and readable command structure
- Interactive loop for continuous input
